# CWeather

## Description
- A weather app created using [openweathermap API](https://openweathermap.org/api)
### More details in the repository: [CWeather](https://github.com/vlsuv/CWeather)
### Video: [youtube link](https://youtu.be/tZBzEiAMpFk)

## Used
- MVC architecture
- UIKit
- Auto Layout
- RealmDatabase
- API Networking with URLSession
- CoreLocation
- CAGradientLayer
- CocoaPods
- Unit Tests

# Hacker-news-reader
## Description
- A hacker news reader for iOS.
### More details in the repository: [hn](https://github.com/vlsuv/hn)
### Video: [youtube link](https://youtu.be/IuCv7aw9vEQ)

## Used
- Hacker News API
- MVVM + Coordinators
- AssemblyBuilder for creating modules
- RxSwift, RxCocoa
- Alamofire
- Without the use of storyboards
- AutoLayout; Xib
- UserDefaults
- SFSafariController
- UIKit
- CocoaPods
